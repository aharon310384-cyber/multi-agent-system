## 🎯 Что искать в потоке (сигналы)

При парсинге источников `intel-scout` должен извлекать:

1. **Новые MCP-серверы** — упоминания `mcp__*`, github-ссылки на MCP
2. **Новые skills** для Claude Code — паттерны, шаблоны
3. **Обновления Claude Code / Anthropic SDK** — версии, фичи
4. **Новые техники промптинга** — конкретные паттерны
5. **Новые agentic-паттерны** — оркестрация, sub-agents, hooks
6. **Альтернативные стеки** — Cursor, Antigravity, Cline (для сравнения)
7. **Кейсы автоматизации бизнеса** — для контента школы
8. **Инструменты для видео-контента** — Remotion, Premiere AI, монтаж

## 🚫 Что фильтровать (шум)

- Хайповые ролики без технического содержания
- Базовые туториалы «что такое ChatGPT»
- Дублирующиеся новости (одно событие в N каналах)
- Реклама курсов и платных сервисов
- Прогнозы и философия без конкретики

## 🔄 План работы

1. **Утром 09:00** — `intel-scout` собирает поток за прошедшие сутки
2. **Анализ** — фильтрация по релевантности к стеку Георгия
3. **Сверка** — что уже есть в `~/.claude/agents/`, `~/.claude/skills/`
4. **Дайджест** — отправка в Telegram через @Octeday_bot:
   - 🔴 СРОЧНО внедрить
   - 🟡 Рассмотреть
   - 🟢 К сведению
5. **Решение** — Опреатор вручную решает что внедрять
6. **Архив** — отчёт сохраняется в `~/Workspace/Онлайн разведка/архив/`
